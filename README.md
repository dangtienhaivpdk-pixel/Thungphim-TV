# RoPhim TV - Ứng dụng xem phim cho Google TV

## Cách build APK (không cần cài Android Studio)

### Phương án 1: Build tự động bằng GitHub (KHUYẾN NGHỊ - Miễn phí)

1. Tạo tài khoản [GitHub](https://github.com) (miễn phí)
2. Tạo repo mới: nhấn nút **+** > **New repository** > đặt tên `rophim-tv` > **Create**
3. Upload toàn bộ folder này lên repo (kéo thả vào trang web)
4. Vào tab **Actions** > chọn workflow **Build RoPhim TV APK** > nhấn **Run workflow**
5. Chờ ~5 phút > nhấn vào workflow vừa chạy > tải file **RoPhimTV-release** về
6. Giải nén → có file `app-release.apk`

### Cách cài APK lên Google TV

1. Trên Google TV: vào **Settings > Device Preferences > Security & Restrictions**
2. Bật **Unknown sources** (Nguồn không rõ)
3. Cài app **Downloader** từ Google Play (miễn phí)
4. Mở Downloader, nhập link file APK của bạn để tải và cài
5. Ứng dụng **RoPhim TV** sẽ xuất hiện trong menu ứng dụng

### Điều khiển
- **Remote TV**: Dùng nút điều hướng để lướt trang
- **Nút BACK**: Quay lại trang trước
- **Nút HOME**: Về trang chủ RoPhim
